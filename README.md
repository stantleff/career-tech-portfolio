## Featured Projects

### Financial Fraud Detection Dashboard
Enterprise-style fraud monitoring and analyst workflow platform built with Flask, Python, SQLite, Pandas, Flask-Login, and Chart.js.

Features include:
- fraud scoring engine
- fraud investigation workflows
- transaction analytics dashboards
- alerts queue
- CSV transaction uploads
- activity log auditing
- analyst notes and status management
- live Render deployment

**Skills shown:** Python, Flask, SQLite, Pandas, authentication systems, fraud-risk logic, dashboard analytics, operational workflows, deployment configuration, data processing, and analyst workflow design.

**Live Demo:** https://financial-fraud-dashboard.onrender.com/

**Repository:** https://github.com/stantleff/financial-fraud-dashboard

### IT Support Ticket Tracker
A full-stack IT operations and support workflow platform designed to simulate the type of internal ticket management system used by technology operations teams.

Features include:
- authentication/login system
- ticket assignment and status tracking
- SLA monitoring
- technician workload management
- analyst notes/comments
- Chart.js analytics dashboard
- CSV reporting exports
- Docker and Render deployment support

**Skills shown:** Python, Flask, SQLite, SQLAlchemy, Chart.js, dashboard analytics, deployment workflows, CRUD operations, systems support workflows, operational reporting, and end-user experience design.

**Live Demo:** https://it-support-ticket-tracker.onrender.com/

**Repository:** https://github.com/stantleff/it-support-ticket-tracker
